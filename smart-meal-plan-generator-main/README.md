SmartLishe is an intelligent web-based meal planning platform designed to help users achieve healthier lifestyles through personalized nutrition recommendations. The platform generates customized meal plans based on user information such as age, weight, height, dietary preferences, health conditions, and fitness goals.

In addition to serving individual users, SmartLishe provides a professional portal where certified nutritionists and gym instructors can monitor clients, create personalized plans, and communicate directly with users.
